## anaconda 仮想環境の構築

### 仮想環境を作る
https://qiita.com/ozaki_physics/items/985188feb92570e5b82d

仮想環境の閲覧
```
conda info -e
```

python3.6の環境構築
```
conda create -n 環境名 python=3.6
```

仮想環境の起動
```
(conda) activate 環境名
```
condaを入れないと動くときと動かない時がある
